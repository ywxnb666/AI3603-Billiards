# 任务背景与目标
在智能决策与控制领域中，复杂物理环境下的精确操作一直是人工智能的重要挑战之一。台球作为一个具有连续动作空间、复杂碰撞动力学以及长期规划需求的经典任务，为智能体的策略学习和空间控制能力提供了理想的平台。
在本次作业中，我们向你提供了一个模拟台球环境。环境包含基础的台球物理、球体间的碰撞行为，以及基于击球动作所产生的结果反馈。你的任务是设计并训练一个台球智能体，使其在面对对手或特定目标配置时能够做出最优击球决策，实现尽可能高的胜率或得分表现。

# 项目要求

可以**两人**一组分工合作，在第16周课堂上进行汇报，预计在16周周末提交项目材料。提交文件需包含如下几个部分：
### 训练相关文件
包含整个训练过程所需的文件，放入文件夹`train`中，训练过程做到可以复现，包含如下内容：
- 训练所需代码，这部分代码可以不拘泥于项目初始代码结构，可以对`agent.py`、`poolenv.py`等进行必要的修改
- Readme文件，说明训练的环境配置、训练命令、超参数设置等
### 测试相关文件
包含整个测试过程所需的文件，放入文件夹`eval`中，测试过程做到结果可以复现，包含如下内容：
- 测试所需代码，这部分只能在发布版本的基础上，修改`agent.py`、`utils.py`，实现新agent的决策过程
- checkpoint等文件，用于新agent的初始化
- Readme文件，说明测试的环境配置、命令、超参数设置等
### 报告
一份项目报告，说明：
- 方法的思路：不仅可以阐述最终实现的方法思路，过往尝试的方案也可以陈述说明，总结这个过程中的经验教训。
- 方法的实现，说明最终方案的实现细节
- 新方法的效果：运行evaluate.py文件后，项目提出的新智能体获得的分数。
- 项目中的分工与每个人的贡献。

**有关BasicAgentPro的说明**：当对BasicAgent的胜率超过88%时，考虑到边际收益递减，为了能为大家提供更大的算法改进空间，展现算法的优越性，我们提供了BasicAgentPro供大家对打测试。当你的agent对BasicAgent的胜率超过88%时，推荐附上与Pro的对打分数。

**报告格式说明**：报告以**个人**为单位进行撰写，即小组内每人均需提交报告，展示对项目的思路和贡献。
报告请使用英文撰写，限制15页以内，陈述清楚即可，不需要在意报告页数，也不推荐大家卷报告页数。最终根据代码完成度、报告内容、击球效果进行综合评分。

**报告模板**：报告模板请参照IEEE会议双栏模板：https://www.overleaf.com/1998687845fyyfzhmpnfkd#b117f2
此为共享链接，各组同学通过**overleaf**打开后左上角点击**Menu**，选择**Copy project**后在自己的模板上编辑

# 环境配置
操作系统推荐 Ubuntu 22.04，Python版本推荐3.13

安装相关依赖：
```bash
# 创建环境
conda create -n poolenv python=3.13
conda activate poolenv
# clone pooltool
git clone https://github.com/SJTU-RL2/pooltool.git
cd pooltool
# 安装poetry
pip install "poetry==2.2.1"
# 基于源代码安装poolenv
poetry install --with=dev,docs
# 运行base agent需要安装的库
pip install bayesian-optimization numpy
```
# 实现思路
本项目不限制实现思路，可以使用包括但不限于以下思路的实现：
- 融入人类先验的其他AI算法等
- 单智能体强化学习，比如PPO、SAC
- 多智能体强化学习，比如MAPPO等
- 基于大语言模型或者多模态大模型的算法
- 基于蒙特卡洛树搜索的算法，比如Muzero
